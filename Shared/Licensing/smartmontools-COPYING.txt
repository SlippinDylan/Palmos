No `smartctl` binary is bundled in the current DrivePulse source or release
pipeline. Consequently, this repository does not currently distribute the
smartmontools work covered by a COPYING file.

If a release begins distributing a smartctl companion, replace this notice with
the verbatim `COPYING` file from the exact smartmontools source release used to
build that companion. The upstream project is available at:

  https://www.smartmontools.org

Do not publish a release claiming bundled smartctl support while this notice is
present. The corresponding release must include the complete applicable GPLv2
license text and verify that the shipped binary and its license are present.
